Ayuda - Ver

Ver Datos

En la sección "Ver Datos" puedes acceder a toda la información relacionada con los registros del instituto. Aquí puedes consultar datos de estudiantes, profesores, asignaturas y otros registros importantes. Esta sección te permite visualizar la información de manera detallada y organizada.

Ver Archivos

En "Ver Archivos" tienes acceso a todos los documentos y archivos relevantes del instituto. Puedes buscar, filtrar y visualizar diferentes tipos de archivos, como documentos PDF, imágenes y otros formatos. Esta sección es esencial para la gestión y consulta de documentos importantes.

Aulas Reservadas

La sección "Aulas Reservadas" te muestra un listado de todas las aulas que han sido reservadas. Aquí puedes ver el estado de las reservas, las fechas y horas de las mismas, así como el propósito de cada reserva. Es una herramienta útil para la gestión y organización de los espacios dentro del instituto.

Clases

En la sección "Clases" puedes ver el horario y la organización de las clases. Esta sección te proporciona una visión clara de las asignaturas, los profesores que las imparten, los horarios y las aulas asignadas. Es fundamental para estudiantes y profesores para gestionar sus horarios y actividades académicas.
